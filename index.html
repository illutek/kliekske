illutek Boilerplate For Drupal8

Boilerplate Drupal8 theme

Dit als een start punt voor het opzetten van een Drupal8 theme, na het uitvoeren van onderstaande nog wel de verwijzingen naar de libraries in het info.yml aanpassen naar de theme naam, na dit alles zou dit een werkende theme moeten opleveren (theme plaatsen in /themes/custom/yourtheme)

Instalatie

Installeer het theme met

git clone git@github.com:illutek/theme-directory.git yourtheme

Naar de directory yourtheme

cd yourtheme

Installeer alle dev dependencies met

yarn install

of

npm install

Hernoem illutek.theme naar yourtheme.theme

mv illutek.theme yourtheme.theme

Hernoem illuetk.libraries.yml naar yourtheme.libraries.yml

mv illutek.libraries.yml yourtheme.libraries.yml

Hernoem illutek.info.yml naar yourtheme.info.yml

mv illutek.info.yml yourtheme.info.yml

Instaleer theme dependencies met bower

bower install

De dist directory en css aanmaken met

yarn start

of met

gulp

Yarn als componenten manager voor dev modules Gulp als automation tool (commando 'yarn start' of met 'gulp' start gulp op ), er wordt een dist folder aangemaakt.
